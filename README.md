# HoardStash - Craft Supply Marketplace

## Overview
A marketplace for buying and selling craft supplies: yarn, fabric, paints, tools, books, and more.

**Live Site:** https://www.hoardstash.com

## Business Model
- Free to list items
- Platform takes 5% of each sale
- Platform provides shipping labels

## Tech Stack
- Next.js 14 + React + TypeScript
- Supabase (PostgreSQL + Auth + Storage)
- Stripe Connect (payments + seller payouts)
- EasyPost (shipping labels)
- Vercel (hosting)

## Key Features
1. User accounts (buyers & sellers)
2. Product listings with photos
3. Search & filter by category, price, location
4. Shopping cart & checkout
5. Stripe Connect for payments
6. EasyPost integration for shipping labels
7. Buyer-seller messaging
8. Reviews & ratings
9. Order tracking
